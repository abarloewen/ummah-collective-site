Activate the booking form

One-time step. This sends a single test submission to FormSubmit, which then emails info@ummah-collective.com a confirmation link. Click that link and the booking form is live for everyone.

Important: open this page on a network where formsubmit.co is not blocked — e.g. your phone on mobile data with Wi-Fi off, or after pausing any ad-blocker / DNS filter. If the button below leads to a "site can't be reached" page, the network is still blocking it.

  1. Tap Activate booking form above.
  2. If FormSubmit shows a reCAPTCHA, complete it.
  3. Open info@ummah-collective.com and click the "Activate Form" confirmation from FormSubmit.
  4. Done — the booking form at /booking.html now delivers to info@ from every network.

Once confirmed you can delete this page. It's hidden from search engines.
